# Plma
지속적으로 wishlist를 작성하여 충동적인 소비에 대한 경고로 MZ 세대에게 긍정적인 소비 습관을 심어주는 것을 목적으로 제작한 앱입니다


---

### 회원가입 및 로그인

- JWT를 활용한 회원가입 및 로그인

### 예산

- 메인페이지
- 예산페이지
- 예산 수정
- 랭킹
- 위시리스트 조회
- 위시리스트 살지 안살지 선택

### 장바구니

- 장바구니 상품 추가
- 장바구니 상품 삭제
- 상품 목록 조회

### 글자인식 조회

- OCR를 활용한 글자 인식

### 기타 API

- 캐릭터 타입 설정
